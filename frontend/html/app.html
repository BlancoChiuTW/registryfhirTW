FHIR 通用轉換工具 本工具可透過設定檔的形式，將任意資料轉換為 FHIR 格式。支援以 Web API、CSV、JSON、Excel 試算表做為資料來源，並可於轉換後自動上傳至 FHIR Server 或與第三方介接。本工具支援 TW Core IG 臺灣核心實作指引，並以 CC 3.0 BY-NC-SA 授權條款釋出。 Lorex <ceo@sita.tech>
